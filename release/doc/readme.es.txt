                                 nnCron LITE
                                 ~~~~~~~~~~~

nnCron LITE es un pequeño pero poderoso automatizador de tareas que
puede lanzar aplicaciones y abrir documentos en un horario o con una
regularidad especificada. nnCron LITE es un perfecto clon, gratis, de
Cron de UNIX, pero para Windows

Algunas de los mas importantes aspectos de nnCron LITE:

  - Se puede instalar como un servicio del sistema o correr como una
    aplicación común.
  - Entiende el formato "cron table" de Unix y se maneja con el fácil
    de editar archivo de texto llamado 'crontab' y es totalmente
    compatible con el 'crontab' de Unix.
  - puede cambiar y usar los variables del entorno
  - puede correr aplicaciones con las autorizaciones de usuario
    vigente.
  - puede se extendido con "plugins" escritos en Forth
  - puede se fácilmente afinado a sus necesidades
  - no consume muchos recursos y puede correr en PC muy viejas
    sin degradar el rendimiento. (Nunca hemos visto al servicio
    nnCron LITE consumir mas de 150K de memoria.)

nnCron LITE es pequeño pero poderoso, fácil de usar.  Si usted
necesita mas que un programador, nnCron LITE es un automatizador de
tareas sofisticado y merece un vistazo. (http://www.nncron.ru/).

---------------------------------------
Requisitos, instalación / desinstalación de nnCron LITE
---------------------------------------
  - IBM PC or compatible
  - Intel Pentium processor o mejor
  - Windows 95/98/ME/NT/2000/XP

Para instalar nnCron LITE haga doble-click con el botón izquierdo
sobre el archivo de distribución. Durante la instalación se le pedirá
que seleccione una carpeta para la instalación.

nnCron LITE guarda sus preferencias en el archivo 'cron.ini'.

---------------------------------------
Correr y parar a nnCron LITE
---------------------------------------

La manera mas fácil de manualmente iniciar a nnCron LITE como un
servicio es de correr 'startcron.bat' en la carpeta raíz de nnCron
LITE.  Para detener el servicio y descargare a nnCron LITE de la
memoria corra a 'startcron.bat'.

Se pude encontrar referencias a los mencionados programas '.bat' in
el grupo de nnCron LITE en el menú de inicio de Windows.
