nnCron LITE is a complete analog of UNIX cron.

The nnCron LITE is a system command scheduler. The nnCron LITE
program executes commands at specified dates and times. You can
specify regularly sheduled commands as described in cron.tab. The
nnCron LITE is a FREEWARE version of nnCron without scripting. The
nnCron LITE is a perfect clone of UNIX system command scheduler for
all Windows (9x/ME/NT/2k/XP). It understands only classic unix crontab
format. nnCron LITE can work as service or as simple application.
